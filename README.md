# plants-alive

## venv usage 
```
python -m venv venv
./venv/Scripts/activate
pip install -r requirements.txt
```

## running the backend server
`uvicorn server:app --reload`

## opening the webapp
right click on "index.html", copy path, then open that in a web browser
